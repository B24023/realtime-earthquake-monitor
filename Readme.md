# リアルタイム強震モニタ＆総合防災情報アプリ

## 概要
防災科学技術研究所（NIED）や気象庁などの外部APIを活用し、現在の震度分布や直近の地震履歴、津波情報をリアルタイムで可視化するWebアプリケーションです。

## 使用技術
- **フレームワーク:** Streamlit
- **主要ライブラリ:** `requests`,`Pillow`
- **利用データ（API）:**
  - 防災科学技術研究所 強震モニタ画像
  - 気象庁 地震情報リスト (JSON)
  - P2P地震情報 API (津波情報)
